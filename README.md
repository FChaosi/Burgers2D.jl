# Burgers2D.jl
A Julia package for solving the 2D Burgers equation for flow (u(x, y, t), v(x, y, t)):

∂u/∂t + u ∂u/∂x + v ∂u/∂y = ν ∇²u,

∂v/∂t + u ∂v/∂x + v ∂v/∂y = ν ∇²v,

on a doubly-periodic domain.
